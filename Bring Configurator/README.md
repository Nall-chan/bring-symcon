## 1. Funktionsumfang

* Konfigurator zum anlegen von Instanzen in Symcon.

## 2. Voraussetzungen

* IP-Symcon ab Version 8.1

## 3. Software-Installation

* Dieses Modul ist Bestandteil der [Bring!-Library](../README.md#2-software-installation). 

## 4. Einrichten der Instanzen in IP-Symcon

Unter 'Instanz hinzufügen' kann das 'Bring! Konfigurator'-Modul mithilfe des Schnellfilters gefunden werden.

![Instanzen](../imgs/inst.png)  

Allgemeine Informationen zum Hinzufügen von Instanzen in der [Dokumentation der Instanzen](https://www.symcon.de/service/dokumentation/konzepte/instanzen/#Instanz_hinzufügen)


__Konfigurationsseite__:  

Wurde der Konfigurator neu erstellt, oder ist die benötigte [Bring! Konto-Instanz (IO)](../Bring%20Account/README.md) nicht verbunden, wird folgende Meldung ausgegeben:  

![Config](imgs/config1.png)  

Erst nach Konfigurieren der [Bring! Account-Instanz (IO)](../Bring%20Account/README.md) und erfolgreichen Anmelden wird eine Liste der im Zugriff befindlichen Listen angezeigt:  

![Config](imgs/config2.png)  

Über das selektieren einer Zeile und den Button erstellen wird eine neue [Bring! List-Instanz](../Bring%20List/README.md) erstellt.
